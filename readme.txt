Make Wordpress menus and widgets hidden or shown.

== Description ==

Hide and show Wordpress menu and widget on admin page for clean and calm UI.

== Installation ==

1. Download the plugin zip package and extract it

1. Put the folder named "apparition" under /wp-content/plugins/ directory

1. Go to the plugins page in your Wordpress admin panel and click "Activate"

1. Under the Appearance > Admin Menu Manager / Widget Manager, manage menu and widgets

1. The last menu, Show Posts Only, toggles menu to show only post and post types.

== Frequently Asked Questions ==

= Will it affect the front pages of my website? =

No. This plugin just change the visibility of menus and widgets on your admin page.
